# CoreMovie
### 제작 기술
- JSP & Servlet, JavaScript, JQuery, RatingJS, HTML5, CSS3, MySQL

### 필수 구현 페이지
- 로그인, 회원 가입, 회원 정보 수정기능
- 게시글 작성 및 수정, 삭제 기능
- 댓글 등록 및 삭제 기능
- 관리자 페이지 (회원 관리, 게시물 관리)
- api를 활용한 영화 순위 정보
- 영화별 상세 정보 및 별점 부여 기능

### 데이터베이스 구조
- userinfo : 유저 정보
- bbs : 게시물 정보
- comment : 댓글 정보
- review : 평점 정보
